# Non-authoritative gold-draft examples

Files here exercise the drafting contract only. They are not pack gold, human-accepted gold,
freeze inputs, or scoring evidence. A valid JSON shape does not make its Korean transcription
true.

`Ux-TMWnmntM.gold.json` is deliberately a dry run: this implementation environment could read
the transcript evidence but could not audition the audio. Its notes carry the required
`[NON-AUTHORITATIVE DRY-RUN FIXTURE]` marker, and the materializer refuses to write such a file
under `bench/packs/`.
